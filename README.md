# Wordle-Game
A Python Implementation of the Wordle Game
# Features
- Classic Wordle rules (6 attempts to guess the 5-letter word)
- Color-coded feedback using terminal colors
  - Green - Correct letter, correct position
  - Yellow - Correct letter, wrong position
- Game analytics automatically recorded in wordle_analytics.csv
  - Timestamp of each guess
  - Guess accuracy and attempt number
- Uses a word list from wordle_no_dupes.json
# Technologies Used 
- Python
- Pandas
- Json
- Randon
- Datetime
- ANSI terminal color codes
